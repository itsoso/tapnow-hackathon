# 《为什么会这样》— TapNow 黑客松动画项目

> 儿童手绘风都市寓言 / 时间异常 / 成长动画 · 12 镜头 · 2'45"–3'15"

一只总是倒霉的小毛球吃下了能"跳过坏运气"的时间豆,却发现被跳过的不只是倒霉,还有努力、眼泪、朋友和快乐。最后,它把被跳过的时间拿回来,长成了一个不再等待好运的小女孩。

**核心表达**:城市可以帮你跳过坏运气,但被跳过的时间,也会带走你成长的证据。
**儿童落点**:没有运气,也能开心地活着。

## 贴题(Urban Glitch 都市异常)

- 扎根现代都市真实场景:学校、便利店、公交站、红绿灯、雨后水坑、家中窗前。
- 异常 = "城市时间系统坏掉",从第 1 秒就出现。
- 异常引发连锁反应:时间豆 → 跳过坏运气 → 代价 → 归还 → 时间回流 → 成长。
- 取胜点是反差:孩子画出来的温柔世界里,城市时间故障了。

## 三铁律

1. 不做成"仙豆童话"。
2. 不做成"单纯变身"。
3. 不做成"考 0 分也没关系"。
