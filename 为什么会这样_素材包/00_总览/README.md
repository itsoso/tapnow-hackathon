# 《为什么会这样》— 项目总览

> 副标题（仅用于提交信息）：**《为什么会这样：被跳过的时间》**
> 正片标题只显示《为什么会这样》。

## 一句话简介

一只总是倒霉的小毛球吃下了能"跳过坏运气"的时间豆，却发现被跳过的不只是倒霉，还有努力、眼泪、朋友和快乐。最后，它把被跳过的时间拿回来，长成了一个不再等待好运的小女孩。

## 类型

儿童手绘风都市寓言 / 时间异常 / 成长动画。

## 核心表达

> 城市可以帮你跳过坏运气，但被跳过的时间，也会带走你成长的证据。

落到儿童表达：**没有运气，也能开心地活着。**

## 贴题逻辑（Urban Glitch 都市异常）

- 作品扎根现代都市真实场景：学校、便利店、公交站、红绿灯、雨后水坑、家中窗前。
- 异常 = "城市时间系统坏掉"，从第 1 秒就出现，不只服务于变身。
- 异常引发连锁反应：时间豆 → 跳过坏运气 → 代价（记忆/努力/快乐被一起跳过）→ 归还 → 时间回流 → 成长。
- 取胜点是反差：**孩子画出来的温柔世界里，城市时间故障了。**
