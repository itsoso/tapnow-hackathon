# 声音设计

> 音乐必须用**可授权、能说明来源**的素材。提交时需要素材来源说明。

## 音效对照表

| 场景 | 声音 |
|---|---|
| 闹钟跳时 | 叮铃 + 咔哒 |
| 红绿灯卡住 | 轻微电子故障声 |
| 时间跳跃 | 倒带 / 闪白 / 咻 |
| 时间豆 | 温柔发光声 |
| 空奖杯 | 声音突然变远 |
| 时间回流 | 雨声、笑声、纸张声一起回来 |
| 雨中跳舞 | 轻柔雨声 + 水花 |
| 结尾 | 温暖轻音乐 |

## 合规提醒

- 音乐、字体、IP、肖像、素材来源都要可说明来源。
- 中文字幕（"下次再试一次""我不想再跳过了""没有运气，也能开心地活着"）后期用字幕/手写图层加，不让 AI 直接生成。
